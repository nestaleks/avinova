# AVINOVA

Адаптивный немецкоязычный сайт для презентации душевых унитазов AVINOVA. Проект включает главную страницу бренда и отдельные страницы моделей S1 и S2.

## Возможности

- адаптивная верстка для компьютеров, планшетов и смартфонов;
- мобильное меню с закрытием по ссылке, клику вне меню и клавише `Escape`;
- каталог моделей AVINOVA S1 и S2;
- динамическое заполнение характеристик и функций моделей;
- интерактивный FAQ;
- анимация появления секций при прокрутке;
- демонстрационная форма запроса консультации;
- плавная навигация по разделам страницы.

## Технологии

- HTML5;
- CSS3;
- JavaScript без сторонних библиотек;
- Google Fonts: Manrope.

Проект не требует сборки или установки зависимостей.

## Запуск

Сайт можно открыть напрямую через файл `index.html`. Для корректной локальной разработки рекомендуется запустить HTTP-сервер:

```bash
python -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

## Структура проекта

```text
avinova/
├── assets/
│   └── avinova-hero.png  # Основное изображение продукта
├── index.html             # Главная страница
├── s1.html                # Страница модели AVINOVA S1
├── s2.html                # Страница модели AVINOVA S2
```

## Страницы

- `index.html` — презентация бренда, моделей, функций, отзывов и FAQ;
- `s1.html` — характеристики базовой модели S1 Essential;
- `s2.html` — характеристики модели S2 Comfort.

## Примечание

Форма консультации работает только на стороне клиента: после отправки она показывает сообщение об успехе, но не передает данные на сервер. Для рабочего приема заявок необходимо подключить API или сервис обработки форм.
